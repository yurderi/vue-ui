# Vue UI-Toolkit

Yet another simple ui toolkit built to work with vue.

## Installation
```
npm i --save @yurderi/vue-ui
```

```javascript
const Vue   = require('vue')
const VueUI = require('@yurderi/vue-ui')

require('@yurderi/vue-ui/assets/less/all.less')

Vue.use(VueUI)

// ... further code
```
